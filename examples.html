flexdashboard Examples

The examples below illustrate the use of flexdashboard with various packages and layouts. If you want to learn more about how the dashboards were created each example includes a link to it’s source code.

htmlwidgets

These examples are dashboards built as regular R Markdown documents (i.e. they are a simple webpage that can be deployed to any web server). Each of them use htmlwidgets to provide direct interaction with charts on the dashboard.

shiny

These examples are dashboards built as Shiny interactive documents (R Markdown with runtime: shiny). Interactivity based on sidebar inputs as well as brushing is demonstrated.
